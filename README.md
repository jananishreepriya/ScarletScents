# 🌹 Scarlet Scents – Luxury Fragrance Website

A modern, responsive fragrance website originally developed during my internship at **Gurumantra4u** and later enhanced through continuous self-learning. Scarlet Scents showcases modern HTML5 and CSS3 practices, semantic web development, responsive design, animations, and an elegant user experience inspired by premium fragrance brands.

## Project Enhancement

After completing my internship, I continued improving my frontend development skills through self-learning. I learned modern HTML and CSS concepts from YouTube educators such as **Balachandar** and **Dave Gray**.

Using the knowledge gained from these resources, I revisited my original internship project (**RedScents**) and enhanced it into **Scarlet Scents**, focusing on cleaner code, improved responsiveness, modern UI design, better accessibility, and maintainable frontend architecture using only HTML and CSS.

### Improvements Made

* Refactored the website using semantic HTML5 elements
* Improved responsive design for mobile, tablet, and desktop devices
* Added CSS variables for reusable styling
* Implemented Flexbox and CSS Grid layouts
* Enhanced UI with hover effects, transitions, and animations
* Improved product card design and visual presentation
* Organized CSS using modern naming conventions
* Increased code readability and maintainability
* Dark theme implementation
